## 상태 짧게 확인하기
git status -s
- status code
  XY
  - x: staging area의 상태
  - y: 현재 디렉토리의 상태
- ?? : untracked
- A : added
- M : modified
- D : deleted
- MM
- AM
- !! : ignored
  --ignored : 보이지 않는 파일의 상태까지 나타냄
- "" : clean

파일 1개 생성 -> 스테이징 -> 수정 -> 커밋 ->
파일 옮기기 (D -> R)
파일 삭제하기(D)
